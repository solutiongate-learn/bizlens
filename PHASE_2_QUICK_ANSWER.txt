❓ WHAT ARE THE TRADE-OFFS?

   NOTEBOOK
   ✅ Interactive, editable, all theory included
   ❌ Larger file, needs Jupyter
   
   MP4 VIDEO
   ✅ Professional, shareable, works everywhere
   ❌ Larger file, can't edit, just watch
   
   GIF
   ✅ Small file, social-media friendly, loops
   ❌ Low quality for long animations, fixed loop
   
   HTML
   ✅ Interactive, lightweight, no server
   ❌ Limited to Plotly visualizations

   └─ BEST CHOICE FOR STUDENTS!

❓ HOW DO USERS ACTUALLY INTERACT WITH THESE?

   NOTEBOOK (Most Interactive)
   User: Click Colab link
   Browser: Opens notebook in Colab
   User: Reads theory cells
   User: Runs code cells
   Animation: Displays inline
   User: Can modify code and re-run
   User: Can zoom/pan in notebook
   User: Can save as MP4/GIF
   
   MP4 VIDEO (Simple Consumption)
   User: Opens file in player
   Player: Shows video controls (play/pause/scrub)
   User: Can pause and scrub timeline
   User: Cannot modify video
   
   GIF (Automatic Loop)
   User: Opens file
   Image viewer: Auto-loops
   User: Can pause by clicking
   User: Cannot modify animation
   
   INTERACTIVE HTML (Web-based)
   User: Opens in browser
   Browser: Shows Plotly controls
   User: Can click Play/Pause
   User: Can zoom (scroll)
   User: Can pan (drag)
   User: Can hover for info
   User: Cannot modify animation
