## 📋 What's Included

### 🆕 NEW FEATURES

#### Process Mining Foundations (Complete Overhaul)
- **Petri Net Generator** — `bl.process_mining.petri_net_from_log()`
  - Extracts formal process models from event logs
  - Generates places, transitions, arcs, initial/final markings
  - Returns Petri net dictionary structure

- **Petri Net Visualizer** — `bl.process_mining.visualize_petri_net()`
  - Uses networkx + matplotlib
  - Blue circles (places), red rectangles (transitions)
  - Spring layout with directional arrows

- **Causal Net Discovery** — `bl.process_mining.causal_net_from_log()`
  - Discovers direct causality between activities
  - Supports min_support filtering
  - Returns activities + causal edges with counts

- **Causal Net Visualizer** — `bl.process_mining.visualize_causal_net()`
  - Interactive Plotly visualization
  - Spring layout for activity relationships
  - Hover tooltips for edge information

- **Alpha Algorithm** — `bl.process_mining.alpha_algorithm()`
  - Automated process discovery from event logs
  - Finds start/end activities
  - Extracts direct succession + causality relations
  - No manual process modeling needed!

#### Educational Notebook
- **Process_Mining_Foundations.ipynb** (NEW)
  - 17 cells covering theory + practice
  - Petri nets, causal nets, workflow nets
  - Alpha algorithm walkthrough
  - Conformance checking (fitness metrics)
  - Practical examples with event logs
  - **Google Colab ready** ✓
