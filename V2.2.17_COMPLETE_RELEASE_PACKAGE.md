## 🚀 QUICK START (15 Minutes)

### 1️⃣ Update Version (5 min)
```bash
bash UPDATE_VERSION_TO_2.2.17.sh
```

### 2️⃣ Build Package (2 min)
```bash
rm -rf dist/ build/ *.egg-info/
python setup.py sdist bdist_wheel
```

### 3️⃣ Upload to PyPI (1 min)
```bash
twine upload dist/*
```

### 4️⃣ Deploy to GitHub (2 min)
```bash
git add .
git commit -m "Release v2.2.17"
git tag -a v2.2.17 -m "BizLens v2.2.17"
git push origin main v2.2.17
```

### 5️⃣ Verify (5 min)
```bash
pip install bizlens==2.2.17
python -c "import bizlens; print(bizlens.__version__)"
```
