│ 1. ANIMATED JUPYTER NOTEBOOK (.ipynb) — EASIEST & MOST FLEXIBLE                               │
├─────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                 │
│  WHERE TO OPEN:                                                                                │
│    ✅ Google Colab        → Click "Open in Colab" button (no install!)                        │
│    ✅ Local Jupyter       → jupyter notebook file.ipynb                                       │
│    ✅ JupyterHub/Cloud    → Upload to cloud platform                                          │
│    ✅ VS Code + Jupyter   → Extensions available                                              │
│    ❌ Terminal            → Cannot display animations                                          │
│                                                                                                 │
│  EASE OF USE:  🟢🟢🟢 VERY EASY                                                               │
│    • User clicks link OR downloads .ipynb                                                     │
│    • Animations play inline in browser                                                        │
│    • Can modify code and re-run                                                               │
│    • Can export to MP4/GIF from within notebook                                               │
│                                                                                                 │
│  FILE SIZE:    5-15 MB                                                                        │
│  BEST FOR:     Learning, education, interactive exploration                                  │
│                                                                                                 │
│  EXAMPLE WORKFLOW:                                                                             │
│    1. User clicks Colab badge                                                                 │
│    2. Notebook opens in browser                                                               │
│    3. Run cell → animation displays                                                           │
│    4. Can modify code and re-run                                                              │
│    5. Can save as MP4: anim.save('video.mp4')                                                │
│                                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                 │
│  WHERE TO OPEN:                                                                                │
│    ✅ VLC Media Player   → macOS/Windows/Linux, free                                          │
│    ✅ QuickTime         → macOS native                                                        │
│    ✅ Windows Media      → Windows native                                                     │
│    ✅ Web browser       → Chrome/Firefox/Safari (modern)                                      │
│    ✅ PowerPoint        → Embed in slides                                                     │
│    ✅ Word/Google Docs  → Embed in documents                                                  │
│    ✅ YouTube          → Upload for sharing                                                   │
│    ✅ Email            → Attach (~5-20 MB per minute)                                         │
│    ✅ Teams/Slack      → Share in messages                                                    │
│                                                                                                 │
│  EASE OF USE:  🟢🟢🟢 VERY EASY                                                               │
│    • Double-click to play                                                                     │
│    • Works on any computer (no software install needed usually)                               │
│    • Can pause, scrub timeline                                                                │
│    • Professional quality                                                                     │
│                                                                                                 │
│  FILE SIZE:    5-30 MB (depending on duration & quality)                                      │
│  BEST FOR:     Presentations, sharing, professional use                                       │
│                                                                                                 │
│  EXAMPLE WORKFLOW:                                                                             │
│    1. Generate from notebook: anim.save('demo.mp4', writer='ffmpeg')                         │
│    2. Share via email/YouTube/Teams                                                           │
│    3. Anyone can play without installing anything                                             │
│    4. Embed in PowerPoint presentation                                                        │
│    5. High quality, professional appearance                                                   │
│                                                                                                 │
│  NOTE: ffmpeg required locally (optional, not needed in Colab)                                │
│        brew install ffmpeg (macOS) | sudo apt install ffmpeg (Ubuntu)                         │

│    • Auto-loops continuously                                                                  │
│    • Works everywhere (Twitter, Slack, email, etc.)                                           │
│    • Smaller than MP4                                                                         │
│                                                                                                 │
