BUG FIXES (5 Total)
================================================================================

BUG #1: Import AttributeError in core.py
   Symptom: bl.describe() crashed
   Root Cause: from . import quality (imported module, not class)
   Solution: from .quality import quality
   Status: ✅ FIXED & TESTED

BUG #2: Variable Unpacking in process_mining.transition_matrix()
   Symptom: 'int' object is not subscriptable
   Root Cause: Incorrect unpacking of value_counts().items() tuples
   Solution: for (from_act, to_act), count in ...items()
   Status: ✅ FIXED & TESTED

BUG #3: Timedelta Serialization in process_mining.timeline_visualization()
   Symptom: Object of type timedelta is not JSON serializable
   Root Cause: Plotly cannot serialize Python timedelta objects
   Solution: Convert to hours: duration.total_seconds() / 3600
   Status: ✅ FIXED & TESTED

BUG #4: Rich Table Rendering in tables.summary_statistics()
   Symptom: "unable to render int" from Rich
   Root Cause: Integer column names from unnamed Series
   Solution: str(col) conversion before add_row()
   Status: ✅ FIXED & TESTED

BUG #5: Pandas Boolean Dtype Incompatibility
   Symptom: statsmodels errors with boolean dtype from get_dummies()
   Root Cause: Newer pandas returns boolean, statsmodels needs float
   Solution: astype(float) before OLS/Logit
   Notebooks Fixed: Linear_Regression, Logistic_Regression, Conjoint_Analysis
   Status: ✅ FIXED & TESTED
