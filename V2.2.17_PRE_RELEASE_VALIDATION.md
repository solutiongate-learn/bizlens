## 🏗️ PHASE 3: BUILD DISTRIBUTION PACKAGES

### Step 1: Clean Old Builds
```bash
# Remove old distribution files and artifacts
rm -rf dist/ build/ *.egg-info/ src/bizlens.egg-info/

# Verify cleanup
ls -la dist/ 2>/dev/null || echo "dist/ directory cleaned"
```

### Step 2: Verify Version Update
```bash
# Confirm version was updated correctly
python -c "import sys; sys.path.insert(0, 'src'); import bizlens; print(f'BizLens version: {bizlens.__version__}')"

# Expected output: BizLens version: 2.2.17
```

### Step 3: Build Distributions
```bash
# Build source distribution and wheel
python setup.py sdist bdist_wheel

# Verify builds were created
ls -lh dist/
```

### Expected Output
```
dist/bizlens-2.2.17-py3-none-any.whl       # Wheel distribution
dist/bizlens-2.2.17.tar.gz                 # Source distribution
```

### Step 4: Test Installation (Optional but Recommended)
```bash
# Create temporary virtual environment
python -m venv /tmp/test_env
source /tmp/test_env/bin/activate

# Install from wheel
pip install dist/bizlens-2.2.17-py3-*.whl

# Test import
python -c "import bizlens; print(bizlens.__version__)"

# Deactivate
deactivate
```

## 🔧 TROUBLESHOOTING

### Version Files Locked Error
```bash
# If files are locked (permission or editor issue):
# 1. Close the files in any open editors
# 2. Wait a few moments
# 3. Try again

# Alternative: Use copy and replace
cp setup.py setup.py.old
sed 's/2.2.16/2.2.17/g' setup.py.old > setup.py
```

### PyPI Upload Issues
```bash
# Check twine version
twine --version

# Upgrade if needed
pip install --upgrade twine

# Verify .pypirc
cat ~/.pypirc

# Test credentials
twine check dist/*
```

### Git Tag Issues
```bash
# Delete tag if created incorrectly
git tag -d v2.2.17
git push origin --delete v2.2.17

# Then recreate
git tag -a v2.2.17 -m "..."
git push origin v2.2.17
```
